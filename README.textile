h2. JavaScript, JSDoc and ExtJS snippets

h3. Install

After you clone, move the folder to ~/Library/Application\ Support/Espresso/TEA/

The result should be similar to this:
<pre><code>~/Library/Application\ Support/Espresso/TEA/UserActions.xml
~/Library/Application\ Support/Espresso/TEA/UserCategories.xml
</code></pre>

Alternatively you can run these commands:
<pre><code>cd ~/Library/Application\ Support/Espresso
git clone git://github.com/rdougan/tea-for-espresso_javascript-extjs-jsdoc.git "TEA"
</code></pre>

h3. About

* ExtJS and JSDoc snippets taken from the ExtJS.tmbundle by edspencer and rdougan.
